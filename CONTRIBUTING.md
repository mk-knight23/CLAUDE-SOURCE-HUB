# 🤝 CONTRIBUTING: The Metadata-Only Vision

Welcome! We are honored that you want to contribute to the **CLAUDE SOURCE HUB**. Together, we are building the world's most definitive, safe, and research-grounded archive for agentic architecture.

## 🛡️ Our North Star: Metadata Only
We have a **strict, non-negotiable policy**: This repository does not host original, proprietary source code. 

**What we DO accept:**
-   **Architectural Synthesis:** Deep-dives into how hidden systems work.
-   **Metadata Manifests:** JSON/YAML mappings of discovered modules.
-   **Clean-Room Concepts:** Pseudo-code or diagrams explaining core logic.
