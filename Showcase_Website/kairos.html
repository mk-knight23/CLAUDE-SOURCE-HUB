KAIROS: The Always-On Proactive Assistant

By ZEUS • 10 min read

The 2.1.84 leak provided more than just a snapshot of a current product; it offered a glimpse into the future of proactive AI. Hidden within the `assistant/` directory was **KAIROS**, an unreleased mode that transforms Claude from a reactive tool into a persistent, always-on development partner.

KAIROS (Ancient Greek for "the right or opportune moment") is built on the philosophy that an AI agent should not wait for user input to provide value. Instead, it should actively observe the environment and intervene when it identifies an optimization or an impending error.

The Heartbeat: 500ms Tick Loop

The core of KAIROS is a high-frequency "Tick" loop. Every 500 milliseconds, the assistant performs a shallow scan of the current working directory, git state, and active terminal buffers. This information is reconciled against its internal state to determine if action is required.

🛠️ Proactive Triggers

  • Error Identification: Detection of failing background tests or build errors.
  • Drafting: Automatically generating commit messages or PR descriptions based on file changes.
  • Documentation Sync: Identifying when code changes have made existing `README.md` or `GUIDE.md` files stale.

Exclusive Toolset

KAIROS-enabled builds have access to tools that are hidden from the standard agent harness. These include `SendUserFile` for proactive asset delivery and `SubscribePR` for monitoring asynchronous review cycles.

The 15-Second Blocking Budget

To ensure that a proactive assistant doesn't become a nuisance, the architecture implements a strict "blocking budget." Any action that would take more than 15 seconds to execute—such as a large-scale refactor—must be queued for user approval, while smaller optimizations are handled silently in the background.

Privacy and "Brief Mode"

Recognizing the potential for terminal noise, KAIROS introduces "Brief Mode," a highly condensed output format that uses minimal characters to signal activity. This ensures that the developer remains the focus of the environment, with the AI acting as a subtle, omnipresent safety net.

As we integrate KAIROS-inspired logic into open-source frameworks like Claw Code, the architectural patterns revealed in the 2.1.84 leak will serve as the definitive blueprint for the next generation of autonomous assistants.
